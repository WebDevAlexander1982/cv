Tips voor studeren (section 1)

  • Breek code
    Probeer de code eerst te begrijpen, niet simpel over te typen. Bekijk een video 10 min, stop, bedenk wat ik heb geleerd, snap ik alles? Probeer dan de code zelf te maken en te schrijven. Lukt dat? Dan door. Zo leer je het snelst. Kopieer geen dingen die je niet 100% begrijpt. Breek het anders in stukjes en probeer elk deel te begrijpen, te manipuleren en kijk wat er gebeurt.
  • Nuttig
    Programmeer om iets te creeren, dat motiveert en leer je het snelst.
  • Internet
    Een verzoek van de client wordt via de internetprovider naar de DNS verstuurd. DNS is groot telefoonboek en zoekt het juiste IPadres, dan wordt dat teruggestuurd. Dan wordt opnieuw een bericht via de provider naar de juiste server gestuurd om de bestanden op te halen.
  • Website
    bestaat uit HTML (structuur), CSS (aankleding) en javascript (functionaliteit)
